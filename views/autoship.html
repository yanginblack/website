Autoship Project

Developed a web application for our staffs to automatically ship orders. It can extract all the satisfied orders, organize and classify them, then purchase ship labels and package lists, update order status and tracking number to our database, and email customer for shipment â€” including fraud customer and duplicate order checking

After log in, user can see the main panel, including the order process section, print section and error check section. It will show how many orders can be processed and shipped, and show the number of orders with special cases which requires manually check.
After clicked autoship, the system will process orders automatically, showing remaining time. Because of the api limitation, this system will process 20 orders a time, after those orders are purchased tracking number and updated database, it will continue another bulk.
After all the orders processed, system will group those orders together and named a batch. User can print this batch anytime they like, print includes shipping label from easyship and packagelist.
